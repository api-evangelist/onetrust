# OneTrust (onetrust)

OneTrust is an enterprise privacy, security, ethics, and ESG platform. Its developer portal exposes APIs across Privacy Management, Third-Party Risk (Vendorpedia), Cookie Consent (Server-Side CMP), Certification Automation, GRC, AI Guard, and a broad SDK suite for mobile, OTT/CTV, and web.

## APIs
- **OneTrust Platform API** — cross-product REST API: https://developer.onetrust.com/onetrust/reference
- **Server-Side CMP API** — consent persistence, banner/preference UI, IAB TCF 2.3 + GPP.
- **OneTrust AI Guard API** — POST /classifications/v1, POST /metric, GET /health.
- **Third-Party Risk (Vendorpedia) API** — vendor risk and assessments.
- **OneTrust SDK Suite** — mobile, OTT/CTV, and website CMP SDKs.
